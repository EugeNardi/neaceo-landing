OPCIÓN 1: DESDE GITHUB (RECOMENDADO)
────────────────────────────────────────────────────────────────────────

1. Crear repositorio en GitHub:
   
   a) Ve a https://github.com y crea un nuevo repositorio
   b) Nómbralo: neaceo-technologies
   c) NO inicialices con README

2. Subir el código:
   
   Abre la terminal en esta carpeta y ejecuta:
   
   git init
   git add .
   git commit -m "Initial commit - NeaCeo Technologies"
   git branch -M main
   git remote add origin https://github.com/TU_USUARIO/neaceo-technologies.git
   git push -u origin main

3. Conectar con Netlify:
   
   a) Ve a https://app.netlify.com
   b) Click en "Add new site" → "Import an existing project"
   c) Selecciona "GitHub" y autoriza
   d) Busca y selecciona "neaceo-technologies"
   e) Configuración:
      - Build command: npm run build
      - Publish directory: out
   f) Click en "Deploy site"

4. Configurar nombre del sitio:
   
   a) Una vez desplegado, ve a "Site settings"
   b) Click en "Change site name"
   c) Ingresa: neaceo
   d) Tu sitio estará en: https://neaceo.netlify.app


OPCIÓN 2: DRAG & DROP (MÁS RÁPIDO)
────────────────────────────────────────────────────────────────────────

1. Construir el proyecto:
   
   Abre la terminal en esta carpeta y ejecuta:
   
   npm run build

2. Subir a Netlify:
   
   a) Ve a https://app.netlify.com
   b) Arrastra la carpeta "out" a la zona de deploy
   c) Espera a que termine el deploy
   d) Ve a "Site settings" → "Change site name"
   e) Ingresa: neaceo
